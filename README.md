# 🌱 Minimalist Business Idea Validator

A clean, shareable startup idea validator built on the **Minimalist Entrepreneur** framework by Sahil Lavingia.

Enter any startup idea and get a structured breakdown:
- **1,000 True Fans** — who they are, where to find them, their pain point, and willingness to pay
- **Community Strategy** — the right platform, existing communities to join, and a go-to-market approach
- **Smallest Viable Product** — core solution, features to build now, features to skip, and tech stack
- **Revenue Model** — pricing strategy, path to $10K MRR, and alternative monetization angles
- **Minimalist Verdict** — a 0–100 score with actionable next steps

Results are shareable via URL — the idea is encoded as a query parameter.
